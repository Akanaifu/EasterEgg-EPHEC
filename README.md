# Easter Egg Challenge - EPHEC

Application de challenge Easter Egg pour l'EPHEC avec backend Node.js/Express et Firebase.

## Installation

1. Installer les dépendances :

```bash
npm install
```

2. Créer un fichier `.env` à la racine (voir `.env.example`)

3. Configurer Firebase Admin SDK (optionnel pour production) :
   - Télécharger le fichier de clé de compte de service depuis Firebase Console
   - Le placer dans le projet comme `serviceAccountKey.json`

## Démarrage

### Mode développement

```bash
npm run dev
```

### Mode production

```bash
npm start
```

Le serveur démarre sur `http://localhost:3000`
